# create-pers-app

Generate a parcel-electron-react-sass based application from https://gitlab.com/eevargas/parcel-electron-react-sass

## Installation

```bash
$ npm install -g create-pers-app
$ create-pers-app my-app
$ cd my-app
$ npm start
```
